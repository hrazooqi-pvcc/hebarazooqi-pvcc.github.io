Networking and Internet
TERM DEFINITION
Network Classification Method of classifying networks.
  • LAN
    • Local area network.
  • MAN
    • Metropolitan area network.
  • WAN
    • Wide area network.
Network models
  • Bus network
    • Devices are connected to bus network.
  • Star network
    • Star network devices are connected to a hub.
Protocols Establish rules so activities are conducted.
  • CSMA/CD Ethernet
    • Dictates messages to be broadcasted on the bus.
  • CSMA/CA Wifi
    • Regulate acess to Wifi networks and avoid collisions.
Combining networks: Extended communication system to form larger network.
  • Repeater
    • Signals back and forth between buses.
  • Bridge
    • Connects bus through destination address.
  • Switch
    • Bridge with multiple connections.
  • Router
    • Connection between networks to form internet.
  • Gateway
    • Point where a network is linked to the internet.
Client/server model Interprocess communication.
  • P2P:peer-to-peer
    • Provides and receive service.
  • Distributed system
    • Consists of software units that execute as processes.
  • Cloud computing
    • Pools of shared computers on network can be allocated.
Internet Addressing Assigns adresses for computer system.
  • Tier 1
    • Network infrastructure that reaches internet.
  • Tier 2
    • Transit from Tier 1 that provide internet's communication.
  • Tier 3
    • Acess to failed routers.
IP address Address of the internet.
  • ICANN - Internet Corporation for Assigned Names and Numbers
  • DNS - Provides address translation.
  • TLD: Top-level domain - Domain classification for commercial institutions.
  • Country-code TLDs
Internet protocols
  • FTP
    • Application for listing and copying files across a network.
  • HTTP
    • Handles webpages.
  • SMTP
    • A way two computers interact when transmitting email messages.
World Wide Web
  • URL
    • Retrieve documents on a web.
  • Client-side activities
    • Activities preformed by a client. Otherwise known as a browers.
  • Server-side activities
    • Activities preformed by a sever or webserver.
Internet four-layer software Provides structure to application software.
  • Application layer
    • Prepares and provides message for address.
  • Transport layer
    • Chops message into packets.
  • Network layer
    • Assigns address to each packet.
  • Link layer
    • Transfers the packets.
Internet is:
  • Packet-switching network
    • Packets are broken into units to addressing information.
  • Fault tolerant
    • Ensure reliability.
  • Redundant
    • The extra components for a system.
Cybersecurity A study that protects networks and computers from an attack.
  • Malware
    • Malicious software.
  • Virus
    • Infects computers by inserting itself into a program.
  • Worm
    • Autonomous program that transfers through a network.
  • Trojan horse
    • Disguised as a program that tricks the victim.
  • Spyware
    • Collects information about computer that relays back to the attack.
  • Phishing
    • As for the information rather spying for it.
  • DDoS attack
    • A technique that overloads the computer with messages.
Protection
  • Firewall
    • Filters traffic passing through a point in the network.
  • Spam filter
    • Firewalls that block unwanted emails.
  • Proxy server
    • Acts as an intermediary between client and server. Shields from unwanted action.
  • Antivirus software
    • Detects and removes presence of virus.
  • Cryptography
    • Study of sending and receiving messages in adversaries.
  • Public key encryption
    • encryption and decryption keys are known while encoded messages are secure
