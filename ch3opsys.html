Operating Systems Terminology
TERM DEFINITION
Operating system Large and complex softwares that manages a computer.
Job Queue Storeage organization that jobs are ordered in.
FIFO First in, first out order for a queue.
Interactive Process Allows program to carry out dialouge.
Time Sharing Operating systems that allow users at the same time.
Multiprogramming Time is divided into intervals then executed in each job.
Application software Programs preform tasks in machines ultilization.
System Software provides structure to application software.
Utlity Software Consists of software units that enhance the capabilities of an operation system.
Graphical user interface Objects are manipulated; files, mouse, icons.
Shell Older interface used to communicate via textual messages.
Kernel Internal portion of operating system that preforms basic functions.
Device Driver Software units that communicate with controllers to carry out operations.
File Manger Maintains records of all files stored in mass storage.
Memory Manager Coordinating the machines memory.
Virtual Memory Large functional memory space made by paging.
Boot Strapping How operating system gets started:transfers the operating system to main memory.
ROM Aids in how a computer turns on: read only memory.
Deadlock Holds data for outlet at a certain time.
Super user Person with high access to operating system.
Multi-Factor Authentication 2 or 3 steps to access private information.
