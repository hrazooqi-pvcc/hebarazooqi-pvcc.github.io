Git/Github Terms

  • Respository- Digital storage to store files and folders.
  • Commit-Finalizing changes.
  • Fork- A process where it copies itself.
  • Push- Collections of processes.
  • Pull Requests-Merging new code changes to respository.
  • Workflows-Managment system to excute a process.
  • Issues-Problems that are tasked to be fixed.
  • Raw button-Opens a file in a raw form.
  • Blame button-Examines the file by the lines.
